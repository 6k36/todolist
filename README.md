🍔 老爹汉堡店 (Papa's Burgeria) - 网页重制版
这是一个基于 HTML5、Tailwind CSS 和 JavaScript 开发的网页小游戏，致敬了经典模拟经营游戏《老爹汉堡店》。本项目作为小组作业开发，实现了从订单接收、肉饼煎烤、汉堡组装到商店升级的完整游戏循环。

🚀 项目启动步骤 (老师必看)
由于本项目采用了纯前端技术栈且包含大量音效与动态素材，请按照以下步骤运行以获得最佳体验：
1. 下载并解压
确保将 GitHub 仓库中的所有文件克隆或下载到本地。
2. 检查目录结构
运行前请确保文件夹内包含以下必要资源（见文件列表截图）：
new.html (主程序文件)
background.jpg (背景图片)
laodie.gif (老爹形象)
table.png (桌台图片)
*.mp3 (包含 start, game, end, shop 四个音效文件)
*_sway.gif (对应成员的动态展示素材)
3. 运行游戏
方法 A ： 使用 VS Code 的 Live Server 插件运行 new.html。
方法 B ： 双击new.html直接在浏览器中打开。
注意： 进入页面后，由于浏览器的自动播放策略，必须点击屏幕一次（点击“开启声音”遮罩层）才能听到游戏背景音乐。
🎮 核心玩法说明
订单系统：左侧实时显示顾客需求，包括配料种类和要求的肉饼熟度。
烤肉台 (Grill Station)：下入肉饼并根据进度条翻面。
🔵 蓝色：生肉
🟢 绿色：完美翻面/起锅点
🟡 橙色：熟肉
🔴 红色：烤糊
组装台 (Build Station)：严格按照 面包底 -> 配料/肉饼 -> 面包顶 的顺序拖拽堆叠。
商店与团队：
每天营业结束后，可进入商店。
使用赚取的金币招募小组成员，每位成员都会为店铺提供强大的 Buff 加成。
🛠️ 技术栈
HTML5 & CSS3：基础架构与动画设计。
Tailwind CSS：响应式布局与美化。
JavaScript (ES6+)：核心逻辑、计时器驱动引擎、拖拽交互。
Canvas Confetti：庆祝特效。
Iconify：图标库支持。
